# 安全部署指南 - 解决"危险网站"警告

## 问题原因

浏览器将您的网站标记为"危险网站"通常是由于以下原因：

### 1. 内容相关
- 网站包含加密货币相关内容
- 使用了"crypto"、"bitcoin"等敏感关键词
- 浏览器的安全扫描误判为金融诈骗网站

### 2. 域名相关
- 使用免费域名服务（.tk, .ml, .ga等）
- 域名包含敏感词汇
- 新注册的域名缺乏信誉历史

### 3. 技术相关
- 缺少安全头部配置
- SSL证书问题
- 部署平台被误判

## 已实施的解决方案

### ✅ 1. 安全头部配置
已在 `next.config.js` 中添加：
- X-Frame-Options: DENY
- X-Content-Type-Options: nosniff
- Referrer-Policy: strict-origin-when-cross-origin
- X-XSS-Protection: 1; mode=block
- Content-Security-Policy: 严格的CSP策略

### ✅ 2. Vercel部署配置
已在 `vercel.json` 中配置：
- 安全头部
- 函数超时限制
- 框架优化

### ✅ 3. 合规性声明
- 创建了 `/security` 页面说明网站用途
- 添加了robots.txt文件
- 在页脚添加了安全合规标识

### ✅ 4. 透明度提升
- 开源代码链接
- 明确的免责声明
- 数据来源说明

## 推荐的额外措施

### 1. 域名选择
```
推荐：使用正规域名注册商
- .com, .net, .org 等主流后缀
- 避免包含"crypto"、"coin"等敏感词
- 选择有信誉的域名注册商
```

### 2. SSL证书
```
确保：
- 使用有效的SSL证书
- 证书来自可信的CA机构
- 配置正确的HTTPS重定向
```

### 3. 网站验证
```
提交到：
- Google Search Console
- Bing Webmaster Tools
- 申请网站安全认证
```

### 4. 内容优化
```
添加：
- 详细的关于我们页面
- 隐私政策
- 使用条款
- 联系方式
```

## 如果仍然出现警告

### 1. 浏览器特定解决方案

**Chrome/Edge:**
```
1. 点击"高级"
2. 点击"继续访问 [网站] (不安全)"
3. 或在地址栏输入：chrome://settings/security
4. 关闭"安全浏览"（不推荐）
```

**Firefox:**
```
1. 点击"高级"
2. 点击"接受风险并继续"
3. 或在地址栏输入：about:config
4. 搜索并修改相关安全设置
```

### 2. 网站验证服务
```
提交到以下服务进行验证：
- Google Safe Browsing
- Norton Safe Web
- McAfee SiteAdvisor
- Sucuri SiteCheck
```

### 3. 联系部署平台
```
如果使用Vercel/Netlify等：
1. 联系客服说明情况
2. 提供网站用途说明
3. 申请白名单添加
```
